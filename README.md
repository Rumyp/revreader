revreader — minimal, focused EPUB reader for terminal and desktop.

Usage

- GUI: ./revreader.py mybook.epub  (or run and press Ctrl+O)
- Terminal: ./revreader.py -t mybook.epub

Install

- pip install -r requirements.txt
